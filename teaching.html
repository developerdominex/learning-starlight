Starlight SLMD → Markdown Tutorial

Welcome! This guide shows how to write .slmd files and convert them to ready-to-use Markdown using the Starlight MD Converter.

1. Writing SLMD

SLMD uses simple tags to structure content, similar to HTML but simplified:

Title of Your Document
Bold text here
Italic text here
Inline code here

Colored text here
Example Link


    

  • List item 1
    • 

  • List item 2
    • 

  • List item 3
    • 







Header 1
Header 2




Row 1, Col 1
Row 1, Col 2

Save this content in a file with the extension .slmd, for example: example.slmd

2. Converting SLMD to Markdown

Use the Starlight MD Converter package. Example program in program.sl:

import { convertFile } from "starlight-mdconverter"

sldeploy "=== SLMD to Markdown Converter ==="

let inputFile = ask("Enter path to your .slmd file:")
let outputFile = ask("Enter path for output .md file:")

convertFile(inputFile, outputFile)

sldeploy "✅ Conversion complete! Markdown file saved."

3. Output

The converter will transform your SLMD tags into standard Markdown. For example:

My Document
Important text
Click here

Will be converted to:

# My Document
**Important text**
[Click here](https://example.com)

4. Tips

  • Use <big1>, <big2>, <big3> for headings.
  • Use <bold>, <italic>, <code> for text styling.
  • Use <color colorname> to indicate colored text (Markdown fallback will show color name).
  • Use <ul> and <li> for lists.
  • Use <table>, <tr>, <td> for tables.
